## Job Market Analyzer

A Python-based mini data pipeline designed to simulate a real-world data source for AI/ML training.
This project collects, processes, and analyzes job market data (Software Engineer, Data Engineer, AI/ML roles) and exposes it in a structured format suitable for downstream analytics or machine learning tasks.

##  Project Overview

The goal of this project is to mimic a simplified **data engineering pipeline**:

- Fetch job data from a public API
- Parse and structure the data
- Clean and prepare it for analysis
- Store it as a dataset (CSV)
- Visualize insights using a dashboard

This simulates how raw data can be transformed into a usable dataset for AI training.
